# RhythmBox

# Getting started
- Install the [.NET Core 2.2 SDK](https://dotnet.microsoft.com/download/dotnet-core/2.2)
- Clone the repository
```
git clone https://github.com/BlauFx/RhythmBox
```

build and run:
```
dotnet run --project RhythmBox.Desktop
```
