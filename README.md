# RhythmBox

# Getting started
- Install the [.NET 6 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/6.0)
- Clone the repository
```
git clone https://github.com/BlauFx/RhythmBox
```

build and run:
```
dotnet run --project RhythmBox.Desktop
```
